Çynergie

Consommation énergétique en temps réel

Le projet

Çynergie a pour but de fournir une interface de visualisation en temps réel de la consommation des ressources énergétiques de l'École de technologie supérieure(ETS). Cela peut inclure la consommation d'eau, d'électricité, de gas, etc.

Technologies utilisées

Prometheus

La plateforme open-source Prometheus est utilisée afin de conserver une base de données historiques à des fins d'analyse sur une plus longue période.

Grafana

Nous utilisons Grafana afin de présenter les données que l'on va chercher dans Prometheus.
